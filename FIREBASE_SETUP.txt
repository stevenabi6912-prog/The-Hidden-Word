Firebase setup for The Hidden Word

IMPORTANT: This project expects js/firebase-config.js to EXPORT the config.
Do NOT paste the Firebase "initializeApp(...)" snippet from the console docs.

Correct format (example):

export const firebaseConfig = {
  apiKey: "...",
  authDomain: "...",
  projectId: "...",
  storageBucket: "...",
  messagingSenderId: "...",
  appId: "...",
  measurementId: "...",
};

If you paste the initializeApp snippet, the app will fail to load and buttons won't work.
